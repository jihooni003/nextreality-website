# NextReality Website

Next.js, TypeScript, Tailwind CSS, Framer Motion, Lucide React, Pretendard, and Inter based website starter.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deployment

This project is ready for Vercel. Add these environment variables before deploying:

- `NEXT_PUBLIC_SITE_URL`
- `NEXT_PUBLIC_FORMSPREE_ENDPOINT`
